# 【制作物】
シニア向けスマホ教室「若葉」Webサイト

## 【目的】
地域のシニアやそのご家族に安心感を与え、電話やLINEでのお問い合わせにつなげること。

## 【使用技術】
- HTML
- Tailwind CSS
- JavaScript

## 【担当範囲】
- サイト構成設計
- UIデザイン
- HTML実装
- Tailwind CSSによるスタイリング
- JavaScriptによる機能実装

## 【実装機能】
- レスポンシブ対応
- 文字サイズ変更機能
- 画像スライダー
- FAQアコーディオン
- LINE相談モーダル
- お問い合わせフォーム
- カウントダウンタイマー

## 【工夫した点】
- シニア層の利用を想定し、視認性と操作性を重視したUIを設計
- 電話・LINEなど利用者が慣れた連絡手段を目立つ位置に配置
- アクセシビリティ向上のため文字サイズ変更機能を実装
- FAQや画像スライダーを活用し、安心感と親しみやすさを表現
- スマートフォン利用を前提としたレスポンシブ対応を実施

## 【改善点】
当初はアクセスマップと駅からの行き方が離れて配置されていました。

しかし、利用者は地図を見ながら行き方を確認することが多いため、
アクセスマップの直下に行き方を配置するレイアウトへ変更しました。

これにより、地図と案内を同時に確認できるようになり、
シニア層でも迷わず情報を把握しやすい構成を意識しました。
